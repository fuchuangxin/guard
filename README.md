### 系统介绍
  系统基于LNMP平台，采用FastAdmin框架开发。整合了XRAY CRAWLERGO AWVS NAXSI等安全工具。实现了安全漏洞扫描、项目信息收集、WAF防护、攻击日志收集、漏洞提醒、漏洞统计等功能。
